# A/B Testing Analysis

## Overview

This project performs a complete A/B testing analysis to evaluate the impact of experimental changes on user behavior and business metrics.

The analysis includes statistical hypothesis testing, confidence intervals, significance testing, and decision-making based on experimental results.

---

## Objectives

* Compare control and treatment groups.
* Measure conversion differences.
* Evaluate statistical significance.
* Support data-driven decision making.

---

## Methods

* Exploratory Data Analysis (EDA)
* Hypothesis Testing
* Confidence Intervals
* Statistical Significance Analysis

---

## Technologies

* Python
* Pandas
* NumPy
* SciPy
* Matplotlib
* Seaborn

## Results

The analysis demonstrates whether the observed differences between experimental groups are statistically significant and provides recommendations based on the findings.
